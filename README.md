# 🎮 Игра «2048»

Классическая логическая головоломка **2048**, написанная на Python с использованием графической библиотеки Tkinter. 
Цель игры - сдвигать плитки на поле 4x4, объединять одинаковые числа и постараться собрать плитку с как можно большим номиналом.

## ✨ Особенности

- 🧩 **Классический геймплей:** поле 4x4, механика слияния плиток.
- 🏆 **Система рекордов:** текущий счет и лучший результат сохраняются в локальный JSON-файл.
- ♾️ **Бесконечная игра:** возможность продолжить партию после достижения заветной плитки 2048.
- 🎨 **Оригинальный дизайн:** цвета плиток максимально приближены к классической версии.
- 🇷🇺 **Русский интерфейс:** все меню и подсказки на русском языке.
- 🚀 **Легковесность:** работает автономно, без интернета и тяжелых зависимостей.

## 🛠 Технологии и инструменты

- **Язык:** Python 3
- **GUI:** Tkinter (графический интерфейс)
- **Хранение данных:** JSON (локальное сохранение рекордов)
